## 格式标准

所有文档遵循 GB/T 9704-2012：
- 眉首：份号 + 密级★期限 + 发文机关 + 发文字号
- 主体：标题（二号小标宋）→ 正文（三号仿宋）→ 落款+印章
- 版记：抄送 + 印发机关 + 日期
- 层级序数：一、（一）1.（1）
